# 🎯 Reaction Speed Game

A fun and interactive game built with **Python** and **Streamlit** to test how fast your reflexes are!
Click the button as quickly as possible after the signal appears and see your reaction time in milliseconds.

---

## 🚀 Features

* ⚡ Real-time reaction speed testing
* 🎮 Simple and beginner-friendly interface
* 📊 Displays your reaction time instantly
* 🔁 Play unlimited rounds
* 🌐 Built using Streamlit for easy deployment

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Time Module**

## ▶️ How To Run The Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/AkshatKaridhal/reaction-speed-game.git
```

### 2️⃣ Open the Project Folder

```bash
cd reaction-speed-game
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```

---

## 🎯 How To Play

1. Start the game
2. Wait for the signal/button to appear
3. Click as fast as possible
4. Your reaction time will be displayed instantly
